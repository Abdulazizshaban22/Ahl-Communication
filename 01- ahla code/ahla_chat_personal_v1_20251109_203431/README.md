# Ahla Chat — Personal v1

واجهة محادثة شخصية ببساطة واتساب، مع دعم:
- **E2EE اختياري** (X25519/NaCl) — التشفير/فك التشفير في المتصفح.
- **مرفقات** (رفع ملفات) مع روابط مباشرة.
- **المؤشرات**: Typing / Read عبر WebSocket.
- **تاريخ** مخزّن في Redis + API للرسائل.
- **واجهة بسيطة** RTL جاهزة.

## التشغيل المحلي
```bash
cd infra
docker compose up -d --build
# البوابة: http://localhost:8089/chat
# API:     http://localhost:8000
# WS:      ws://localhost:8089/ws
```

> الميزة E2EE تجريبية لأغراض الاختبار (مفتاح الجهاز محفوظ محليًا). في الإنتاج استخدم تبادل مفاتيح عبر دعوة/QR.

# AWS WAF — Bot Control (TARGETED vs. COMMON)

- **TARGETED**: يغطي نقاط حرجة (تسجيل/دخول/واجهات API) بصرامة أعلى وتكلفة أقل من الشامل.
- **COMMON**: يغطي طيفًا أوسع من الأنماط — مناسب للمواقع العامة ذات المرور الكبير.
- في هذه الحزمة فعّلنا **TARGETED** على Web ACL الخاص بـCloudFront. تستطيع تغيير المستوى لاحقًا بإبدال قيمة `inspection_level`.

> ملاحظة: يفضّل اختبار القواعد في وضع Count أولًا ثم التحويل إلى Block لتفادي الحجب الإيجابي الكاذب.

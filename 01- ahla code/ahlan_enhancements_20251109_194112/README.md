# منظومة أهلا للتواصل — دليل التشغيل السريع

> ملف مُولّد تلقائيًا (20251109_194112). يُكمل وثائق المشروع الحالية ويقترح تشغيلًا موحّدًا.

## المتطلبات
- Docker + Docker Compose
- Node.js 18+/pnpm أو Python 3.11+ (بحسب الخدمات)
- مفاتيح البيئة عبر `.env` (انظر `.env.example`)

## التشغيل السريع
```bash
cp .env.example .env  # حرّر القيم
docker compose up -d  # أو docker-compose
```

## التطوير
- Lint/Format: Prettier (+ ESLint إن تم تفعيله)
- مراقبة: يُنصح بإضافة Prometheus/Grafana إن وُجدت خدمات طويلة

## الأمن والامتثال
- تفعيل Helmet/CSP وسياسات CORS
- إدارة أسرار عبر متغيّرات/أدوات Vault (عدم دفع .env للمستودع)
- PDPL: لافتات الموافقة وسجل الوصول وتصدير/حذف بيانات المستخدمين عند الطلب

## CI/CD (مقترح)
- فحص بناء ولينت واختبار
- بناء صور Docker ونشرها إلى registry
- ترحيل قواعد البيانات تلقائيًا (إن وجدت)

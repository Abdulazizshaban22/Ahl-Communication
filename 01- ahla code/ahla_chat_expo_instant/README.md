# Ahla Chat — Expo Instant Demo
Build: 2025-10-20T07:26:58.120829Z

تشغيل فوري لتطبيق جوال (iOS/Android/Web) بدون أي إعدادات أصلية:
1) ثبّت المتطلبات: Node 18+ و npm أو yarn.
2) داخل المجلد:
   ```bash
   npm install
   npm start
   ```
3) اختر:
   - i: تشغيل iOS Simulator (لو عندك Xcode)
   - a: تشغيل Android Emulator (لو عندك Android Studio)
   - أو امسح QR في Expo Go على جوالك.

> هذا الديمو يعمل **فورًا** بدون Rust/FFI. لاحقًا نبدّل echo المحلي بربط `ahla-core` (Phase A).

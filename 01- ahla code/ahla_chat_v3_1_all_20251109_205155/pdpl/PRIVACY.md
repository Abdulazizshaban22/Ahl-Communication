# سياسة الخصوصية — Ahla Chat v3.1
- يدعم التطبيق إشعارات Web Push بمفاتيح VAPID، ويمكنك إيقاف الاشتراك متى شئت.
- النسخ الاحتياطية تُشفّر محليًا بكلمة مرور (AES‑GCM + PBKDF2) قبل تنزيلها.
- المقاطع تُخزَّن محليًا على الخادم، وتُولَّد صور مصغّرة تلقائيًا.
- لا تُبنى ملفات تعريف سلوكية دون موافقة صريحة.
